## Notes pour l'ACE Pro

- **Purge maligne** : les 5 modèles [MULTI] (#3, #5, #7, #14, #15) sont multicolores par **pièces séparées** ou par zones — c'est le sweet spot de l'ACE Pro (mono-buse). Éviter les modèles multicolores avec changements de couleur dans la même couche sur tout le print.
- **Sans supports** : #1, #2, #3, #4, #5, #8, #9, #10, #13 (supportless revendiqué), #14 — quasi toute la liste.
- **Profils prêts** : les modèles MakerWorld ont des profils 3MF (pensés Bambu, mais paramètres transposables dans Anycubic Slicer Next / OrcaSlicer).
- **Marble runs (#6, #7)** : gros volume d'impression — imprimer module par module, parfait pour écouler des fins de bobines (un module = une couleur).
- **Bateau de bain (#12)** : imprimer le benchmark d'engrenages fourni AVANT le bateau complet pour valider les tolérances de la Kobra S1.
