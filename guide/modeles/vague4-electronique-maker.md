## Notes d'impression

- **PLA partout** sauf si l'objet chauffe : le **Stealth Press 1S (#3)** et le **Heat Set Stand (#4)** tiennent un fer chaud à proximité → privilégier **PETG/ASA** pour les pièces exposées à la chaleur (la doc de chaque modèle précise les pièces concernées).
- **Snap-fit / print-in-place** : #1, #9, #10, #11, #12, #13, #14 sortent sans supports. Pour les boîtiers, imprimer le **fond face plateau** pour les clips nets.
- **`[MULTI]` potentiel** : les organiseurs (#6, #7) et le marquage des étiquettes/tiroirs gagnent à passer le texte/relief dans une 2e couleur → lisibilité maximale, changement à la hauteur = purge minime. Le reste : mono-couleur pour économiser.
- **Établi cohérent** : #6 + #8 forment une base d'organisation Gridfinity-compatible ; #3 + #4 + #5 couvrent toute la soudure ; #9→#12 protègent les cartes des projets en cours.
